# AgoraMesh Documentation

This directory contains the design, planning, and specification documents for AgoraMesh.

## Layout

- `adr/` — Architecture Decision Records.
- `checkpoints/` — Phase completion and planning checkpoints.
- `specs/` — Protocol specifications (object schemas, message exchange, identifiers, etc.).
- `archive/` — Superseded drafts and historical notes kept for reference.
